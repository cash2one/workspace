- If you select 'starts with' option then it searches parts whose part number starts with text entered in part number field.

- If you select 'contains' option then it searches parts whose part number contains entered text somewhere.
- If you select 'Exact match' option then search result contains parts whose part number exactly equals to entered text.

In Description field you can enter a small description of part, technical specification of part, product series, trademark of part and select any of the following options:
- Any of the words, searches parts with matching any one of word from description.
- All the word, searches parts with matching all the words from description.
- None of the word - searches parts which does not match either of the word from description.
- Exact phrase, searches parts with matching exact phrase from description.

In supplier field you can select the supplier from list, which searches all the parts of selected supplier.

You can filter the parts result by using New Items, RoHS Compliant Items, Featured Items, Clearance Items checkboxes.
